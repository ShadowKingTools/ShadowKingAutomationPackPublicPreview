# Install / Setup

## Requirements
- Windows 10/11
- PowerShell 5.1+ (default on Windows) or PowerShell 7+
- This repo downloaded (ZIP) or cloned

## Recommended folder
Extract the pack to a stable path like:
- `D:\ShadowKing_Automation\ShadowKingAutomationPack`
or
- `C:\Tools\ShadowKingAutomationPack`

Avoid running from inside a temporary folder.

## Run (recommended)
Open PowerShell in the pack folder and run:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\RunPack.ps1
